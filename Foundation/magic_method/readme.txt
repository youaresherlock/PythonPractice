魔法方法
__new__: 真正的构造函数, 在生成对象时调用 负责对象的创建
__init__: 初始化函数 负责对象的初始化
__del__: 析构函数,释放对象时使用 del只会将对象的引用计数器减1, 为0才会调用__del__
__str__: 打印对象会打印__str__返回的字符串, 否则打印对象的内存地址
__repr__: 打印, 转换
__setitem__: 按照索引赋值
__getitem__: 按照索引获取值
__len__: 获得长度
__cmp__: 比较运算
__call__: 实现了__call__的类的对象是可调用对象callable object
__add__: 加运算
__sub__: 减运算
__mul__: 乘运算
__div__: 除运算
__mod__: 求余运算
__pow__: 幂
